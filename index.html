Api หน่วยเลือกตั้ง สมาชิกสภาผู้แทนราษฎร พ.ศ. 2566

ข้อมูลต่อไปนี้ถูกแปลงให้อยู่ใน Google Sheet format และ Json โดยข้อมูลตั้งต้นเอามาจากเว็บ data.bangkok
ที่ทางกรุงเทพมหานครได้เปิดเผย ข้อมูลหน่วยเลือกตั้ง สมาชิกสภาผู้แทนราษฎร พ.ศ. 2566 ไว้ ณ วันที่ 11 พฤษภาคม 2566
* โดยข้อมูลนี้อาจมีการเปลี่ยนแปลง กรุณาระมัดระวังในการนำไปใช้ต่อ
